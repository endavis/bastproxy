About

This plugin is a simple substition plugin

Commands

ssub

add


usage: #bp.ssub.add [-h] [original] [replacement]

add a simple substitute

positional arguments:
  original     the output to substitute
  replacement  the string to replace it with

optional arguments:
  -h, --help   show help

clear


usage: #bp.ssub.clear [-h]

clear all substitutes

optional arguments:
  -h, --help  show help

list


usage: #bp.ssub.list [-h] [match]

list substitutes

positional arguments:
  match       list only substitutes that have this argument in them

optional arguments:
  -h, --help  show help

remove


usage: #bp.ssub.remove [-h] [substitute]

remove a substitute

positional arguments:
  substitute  the substitute to remove

optional arguments:
  -h, --help  show help

Base

api


usage: #bp.ssub.api [-h] [api]

list functions in the api

positional arguments:
  api         api to get details of

optional arguments:
  -h, --help  show help

help


usage: #bp.ssub.help [-a] [-c] [-h]

show help info for this plugin

optional arguments:
  -a, --api       show functions this plugin has in the api
  -c, --commands  show commands in this plugin
  -h, --help      show help

reset


usage: #bp.ssub.reset [-h]

reset the plugin

optional arguments:
  -h, --help  show help

save


usage: #bp.ssub.save [-h]

save the plugin state

optional arguments:
  -h, --help  show help

set


usage: #bp.ssub.set [-h] [name] [value]

change a setting in the plugin

if there are no arguments or 'list' is the first argument then
it will list the settings for the plugin

positional arguments:
  name        the setting name
  value       the new value of the setting

optional arguments:
  -h, --help  show help

stats


usage: #bp.ssub.stats [-h]

show plugin stats

optional arguments:
  -h, --help  show help
