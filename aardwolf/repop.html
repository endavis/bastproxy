About

$Id$

This plugin runs gmcp commands after connecting to aardwolf

Commands

Base

api


usage: #bp.repop.api [-h] [api]

list functions in the api

positional arguments:
  api         api to get details of

optional arguments:
  -h, --help  show help

help


usage: #bp.repop.help [-a] [-c] [-h]

show help info for this plugin

optional arguments:
  -a, --api       show functions this plugin has in the api
  -c, --commands  show commands in this plugin
  -h, --help      show help

reset


usage: #bp.repop.reset [-h]

reset the plugin

optional arguments:
  -h, --help  show help

save


usage: #bp.repop.save [-h]

save the plugin state

optional arguments:
  -h, --help  show help

set


usage: #bp.repop.set [-h] [name] [value]

change a setting in the plugin

if there are no arguments or 'list' is the first argument then
it will list the settings for the plugin

positional arguments:
  name        the setting name
  value       the new value of the setting

optional arguments:
  -h, --help  show help

stats


usage: #bp.repop.stats [-h]

show plugin stats

optional arguments:
  -h, --help  show help

Settings

channel


the channel to send the repop message

format


the format of the message
