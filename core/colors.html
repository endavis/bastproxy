About

$Id$

This plugin handles colors

Color Codes: xterm 256 @x154 - make text color xterm 154 @z154 - make background color xterm 154

regular ansi: regular bold Red @r @R Green @g @G Yellow @y @Y Blue @b @B Magenta @m @M Cyan @c @C White @w @W Reset @k @D

Commands

Base

api


usage: #bp.colors.api [-h] [api]

list functions in the api

positional arguments:
  api         api to get details of

optional arguments:
  -h, --help  show help

example


usage: #bp.colors.example [-h]

show color examples

optional arguments:
  -h, --help  show help

help


usage: #bp.colors.help [-a] [-c] [-h]

show help info for this plugin

optional arguments:
  -a, --api       show functions this plugin has in the api
  -c, --commands  show commands in this plugin
  -h, --help      show help

reset


usage: #bp.colors.reset [-h]

reset the plugin

optional arguments:
  -h, --help  show help

save


usage: #bp.colors.save [-h]

save the plugin state

optional arguments:
  -h, --help  show help

set


usage: #bp.colors.set [-h] [name] [value]

change a setting in the plugin

if there are no arguments or 'list' is the first argument then
it will list the settings for the plugin

positional arguments:
  name        the setting name
  value       the new value of the setting

optional arguments:
  -h, --help  show help

show


usage: #bp.colors.show [-h]

show colors

optional arguments:
  -h, --help  show help

stats


usage: #bp.colors.stats [-h]

show plugin stats

optional arguments:
  -h, --help  show help

API

convertcolors


colors.convertcolors(tstr)

    convert @ colors in a string


original defined in plugins/core/colors.py

lengthdiff


colors.lengthdiff(colorstring)

    get the length difference of a colored string and its noncolor equivalent


original defined in plugins/core/colors.py

stripcolor


colors.stripcolor(text)

    strip olors


original defined in plugins/core/colors.py

iscolor


colors.iscolor(color)

    check if a string is a @ color, either xterm or ansi


original defined in plugins/core/colors.py

ansicode


colors.ansicode(color, data)

    return an ansicoded string


original defined in plugins/core/colors.py

stripansi


colors.stripansi(text)

    strip all ansi from a string


original defined in plugins/core/colors.py

colortohtml


colors.colortohtml(input)

  convert color codes to html


original defined in plugins/core/colors.py

convertansi


colors.convertansi(text)

    convert ansi color escape sequences to olors


original defined in plugins/core/colors.py
